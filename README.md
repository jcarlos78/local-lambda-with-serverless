
# local-lambda-with-serverless

This project is a blank lambda serverless project that runs locally

### Prerequisites


*[NodeJs](https://nodejs.org/en/download/)

*Serverless Framework
```
npm install -g serverless
```

### Installing

```bash
#First time useing

npm install -g serverless

npm install
```

## Running the tests

```bash
#Seploy changes
sls deploy

#Call function
sls invoke -f main

#Start local offiline
sls offline start
```
